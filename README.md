# sql-data-cleaning
SQL data cleaning practice project
This is an educational project on data cleaning and preparation using SQL.  
The original dataset in CSV format is uploaded to this repository.  
In this project, I create a SQLite database, import the dataset, make a cleaned copy of the table, and apply data cleaning techniques such as trimming spaces, standardizing text case, validating age values, and removing duplicates.

## Files in this repository
- `club_member_info.csv` – original dataset
- `club_members.db` – SQLite database
- `README.md` – project documentation
